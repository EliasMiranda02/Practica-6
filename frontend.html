FRONT - END

¿Qué es Front - end?

El front end es aquello que ven los usuarios e incluye elementos visuales, como botones, casillas de verificación, gráficos y mensajes de texto. Permite a los usuarios interactuar con la aplicación.


  • HTML
  • CSS
  • JS

Fundamentos del Front end

El desarrollo frontend, también conocido como desarrollo de UL, se refiere a la creación de interfaces de usuario web, las partes de la aplicación que el usuario ve y con las que interactúa. El desarrollo frontend se sustenta firmemente en tres pilares: HTML para el marcado, CSS para el estilo y JavaScript para la lógica y las interacciones. Aunque estas tres tecnologías resistieron la prueba del tiempo, las herramientas y los marcos subyacentes cambian constantemente. Esta sección le brindará un punto de partida para comenzar a explorar el ecosistema frontend.
